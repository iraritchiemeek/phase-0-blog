Map, Map!

In ruby the method 'Map' apply's code to each seperate thing in an array. For example if you wanted to add something onto the end of each element in your array streams =[karearea, kakapo, hihi] streams.map {|stream| puts stream + ' is a type of bird'} would give you karearea is a type of bird kakapo is a type of bird hihi is a type of bird. 'Map!' does the same except instead of returning a new array it edits the original.

The difference betweens arrays and hashes

In Ruby an array is just like a list, it is data stored in an order. You can look up each data entry by using the number it comes on the list. For example if our list was 'Red, White, Blue', colour_array[0] would give you "Red" and colour_array[1] would give you "White"

Hashes are very similar to arrays, however instead of using a number to look up each thing on the list you use a unique name called a key that you have assigned to each list item. For example if you were to make the list 'Musician - Michael Jackson, Actor - George Clooney, Artist - Pablo Picasso' famous_hash["Musician"] would give you "Michael Jackson" and famous_hash["Artist"] would give you "Pablo Picasso".
